🔧 How to replicate this setup

This is a live case study: here's how you can customize the environment for your own lab or classroom.

📦 Client-side storage

Auto-saves to localStorage. No backend needed — works on any static host.

JSON.parse(localStorage.getItem("mt_sessions")||"[]")

📊 Google Sheets sync

Set APPS_SCRIPT_URL in trainer.js. Sessions POST on completion via Apps Script.

APPS_SCRIPT_URL="https://script.google.com/.../exec"

🔑 Sheet credentials

Change SHEET_ID in trainer.js. Sheet must be "Anyone with link can view".

const SHEET_ID="your-sheet-id";

🧬 Cross-domain reading

This page reads from Sheets via gviz API. Same pattern works for any site.

fetch(SHEET_URL) // gviz/tq?tqx=out:json
