🔧 How to replicate this setup

Customize the environment for your own lab or classroom with minimal code changes.

📦 Client-side storage

Auto-saves to localStorage. No backend needed.

JSON.parse(localStorage.getItem("mt_sessions")||"[]")

📊 Google Sheets sync

Set APPS_SCRIPT_URL in trainer.js. Sessions POST on completion.

APPS_SCRIPT_URL="https://script.google.com/.../exec"

🔑 Sheet credentials

Change SHEET_ID in trainer.js. Sheet must be "Anyone with link can view".

const SHEET_ID="your-sheet-id";

🧬 Cross-domain reading

This page reads from Sheets via gviz API. Same pattern works anywhere.

fetch(SHEET_URL) // gviz/tq?tqx=out:json
